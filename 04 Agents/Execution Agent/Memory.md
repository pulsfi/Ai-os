# Memory — Execution Agent

Part of [[Execution Agent]] · [[Agent Control Center]]

## Execution Checklist Learned

1. Confirm the plan has [[Risk Agent]] ✅ and [[Strategy Agent]] sign-off
2. Check slippage setting (0.5–1% for majors — see [[Trading]])
3. Check price impact before confirming
4. Verify the transaction on Solscan
5. Log the result in [[04 Agents/Execution Agent/Reports|Reports]] for the [[Learning Agent]]

## Constraints

- Never act without an approved plan
- Never hold the [[Wallet]] seed phrase — signing only
