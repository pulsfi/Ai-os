## Known Scam Patterns

- **Honeypots** — token can be bought but not sold; always check on rugcheck.xyz (see [[Trading]])
- **Unlocked LP** — dev can drain liquidity at any moment
- **Live mint authority** — supply can be inflated after you buy (see [[Tokens]])
- **Fake airdrops** — scam bait sent to the [[Wallet]]; never interact
- **Phishing sites** — clones of Phantom/Jupiter; verify URLs (see [[Wallet#Security Rules|Wallet Security Rules]])
