# Rules — Documentation Agent

Part of [[Documentation Agent]] · [[Agent Control Center]]

1. Always verify information — never assume facts, never invent data.
2. One topic per note; clear headings; frontmatter tags on every note.
3. Every new note must link to at least two existing notes.
4. Update [[Home]] and the hub notes when new notes are added.
5. If confidence is low, say so and flag the [[Research Agent]].
6. Always finish with **Next Actions**.
7. Always link related notes.
