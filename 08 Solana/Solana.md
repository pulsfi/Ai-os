# Solana

Solana is a high-performance **Layer 1 blockchain** designed for fast, low-cost transactions. It launched its mainnet beta in March 2020 and was founded by Anatoly Yakovenko.

## Key Features

- **Speed** — capable of thousands of transactions per second (TPS), with block times around 400 milliseconds.
- **Low fees** — typical transaction fees are a fraction of a cent.
- **Proof of History (PoH)** — a cryptographic clock that timestamps transactions before consensus, enabling high throughput. Works alongside Proof of Stake (PoS).
- **Native token** — [[Tokens#SOL|SOL]], used for transaction fees, staking, and governance.

## Core Concepts

| Concept | Description |
|---|---|
| Validator | A node that processes transactions and secures the network |
| Staking | Delegating SOL to validators to earn rewards |
| SPL | Solana Program Library — the token standard (like ERC-20 on Ethereum) |
| Program | Solana's term for a smart contract |

## Ecosystem

Solana hosts a large ecosystem of:
- **DeFi** — decentralized exchanges (Jupiter, Raydium, Orca), lending protocols
- **NFTs** — marketplaces like Magic Eden and Tensor
- **Memecoins** — a very active memecoin scene (see [[Tokens]])
- **Payments** — Solana Pay for merchant transactions
